# Product Overview

Claude Code 向けのプラグインマーケットプレイスリポジトリ。コミュニティが作成した再利用可能なプラグインやスキルを配布・インストールするための仕組みを提供する。

## Core Capabilities

- **プラグイン配布**: マーケットプレイスカタログを通じた Claude Code プラグインの集中配布
- **スキル提供**: ユーザーが呼び出し可能なスキル（コマンド）としての機能提供
- **Spec-Driven 開発**: Kiro スタイルの仕様駆動開発による品質管理されたプラグイン開発プロセス

## Target Use Cases

- Claude Code ユーザーが共有プラグインを簡単に発見・インストールする
- 開発者がプラグイン・スキルを構造化されたプロセスで作成・公開する
- チームで共通のワークフローをスキルとして標準化・共有する

## Value Proposition

- Claude Code のネイティブプラグインシステムに準拠した標準的な配布形式
- 仕様駆動開発（Requirements → Design → Tasks → Implementation）による段階的な品質管理
- マーケットプレイスを通じた一元的なプラグイン管理とバージョン管理
